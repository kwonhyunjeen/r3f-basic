# 🧱 r3f-basic

React Three Fiber(R3F)의 핵심 개념들을 학습합니다.

## 📚 소개
이 프로젝트는 [React Three Fiber 공식 문서](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction)를 기반으로, 기초적인 구성 요소부터 `@react-three/drei`, `@react-three/fiber` 등 다양한 유틸리티의 사용법을 실습합니다.
> 📎 개인 학습 페이지: [블로그 바로가기](https://kwonhyunjin.super.site/16013c6d177280e3a4a8d6d459373558)


## ✨ 주요 학습 내용

- React Three Fiber 기본 구조 및 Canvas 설정
- Scene 구성 요소
- Camera 설정 및 OrbitControls 적용
- `useThree` 훅을 통한 Scene, Camera, Renderer 접근
- Object 3D 개념
- Mesh, Geometry, Material 조합을 통한 3D Object 생성
- Leva를 통한 실시간 GUI 조작 패널 구성
- `useFrame` 훅을 통한 애니메이션 구현
- Light, Shadow 설정을 통한 입체감 표현
- 사용자 인터랙션: OrbitControls, 마우스 이벤트

## ▶️ 실행 방법
1. 이 저장소를 클론합니다.
```bash
git clone https://github.com/kwonhyunjeen/r3f-basic.git
cd r3f-basic
```

2. 패키지를 설치합니다.
```bash
npm install
```

3. 개발 서버를 실행합니다.
```bash
npm run dev
```

4. 브라우저에서 `http://localhost:5173`에 접속합니다.
